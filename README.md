# dotfiles
my backup

## to use

1. clone repo into home directory
```
git clone https://github.com/ylyma/dotfiles.git
```

2. go into `dotfiles`
```
cd ~/dotfiles/
```
3. go into `init.sh` and replace [password] with actual password
4. make `init.sh` executable & execute
```
chmod +x init.sh
./init.sh
```
